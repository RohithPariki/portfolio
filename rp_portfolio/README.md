# Developer Portfolio

A modern, minimal, and professional developer portfolio website.

## Setup

No installation is required. This is a static website built with HTML, CSS, and JavaScript.

## How to Run

Simply open the `index.html` file in any modern web browser.

## Customization

1.  **Content**: Edit `index.html` to update your name, links, summaries, and project details.
2.  **Styles**: Modify `style.css` to change the color scheme or layout. Variables are defined at the top of the file.
3.  **Scripts**: `script.js` handles the simple logic for mobile navigation and scroll animations.

## Technologies Used

-   HTML5
-   CSS3 (Custom properties, Flexbox, Grid)
-   JavaScript (ES6+)
-   FontAwesome (Icons)
-   Google Fonts (Inter)
